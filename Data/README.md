## 🚀 Getting Started

### Prerequisites

- Python 3.8+ with packages listed in `requirements.txt`
- MATLAB R2023a+ with Simscape Multibody (optional – required only for the high‑fidelity plant)
- PostgreSQL 13+ (or any PostgreSQL instance)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Sm-bello/AEROTWIN-Dornier228-LandingGear-Dataset.git
   cd AEROTWIN-Dornier228-LandingGear-Dataset
